Development of New Geo-referenced Content in Street Level

As it is required high quality geo-referenced content to express real world authentically in a virtual space, the content aims to three objectives written below:

1. The content should include textures projected from real world preserving original features.

2. The content should provide 3D geometric information about the objectified model.

3. The terrain features attributes such as GIS model should be organically linked with the content.

Such content with those features is referred to as '3D textured GIS model' in this research.

The traditional way of editing the 3D model is to extract 3D geometric information to objectify the model and to map the texture elaborately. Such model is well integrated, though this approach is costly and time consuming. In addition, the users are informed of this model in a 2D projection, which leads to ineffectiveness.

We thus propose the new content as a solution to overcome those limitations. It is able to determine the 3D coordinates of given object in real world using the 3D geometric attributes. At the same time, it is able to link GIS database with the content much closer and flexible. The content includes various features of geo-referenced images, with a range of application indoor to outdoor street level.

The components of 'new media geo-referenced content' are shown above. The content includes direction vector information and depth (or distance) information as well as texture information. Such information is applied to calculating 3D absolute coordinates from the whole area of the content. The coordinates obtained from the content is linked to GIS database as a reference key to bring other attributes to the content.

As shown above, users are able to extract 3D geometric model from the content, and vice versa (referred to as 'augmentation'). It is able to construct 3D models in an economical way with such feature. In addition, the construction and the renewal of GIS database is much economical and effective using the content, since it is able to link GIS database with the content much closer.

The software architecture for proposed new media geo-referenced content is shown in _Fig. 14._ The software consists of the main module and the sub module supporting the main module. The main module interacting with the GIS database consists of '3D coordinates determination module', 'Measurement module', '3D geometric model construction module', and 'Content augmentation module'. Other supporting modules include adjustment computation, coordinate conversion, calibration, visualization, and input/output conversion module.
